# Seb Lettering

Un sello que certifica la conexion entre autor, historia y lector. Aqui las memorias no se pierden; se convierten en legado.

## Desarrollo

Instala dependencias:

```bash
npm install
```

Ejecuta el servidor local:

```bash
npm run dev
```

Abre [http://localhost:3001](http://localhost:3001) en el navegador.

## Scripts

```bash
npm run dev
npm run build
npm run start
```
